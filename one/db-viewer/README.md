# 数据库查看工具

一个简洁美观的Web界面数据库查看工具，基于Python Flask + HTML。启动时自动连接数据库并缓存数据到内存。

## 功能特性

- ✓ 启动时自动连接数据库并缓存所有表数据
- ✓ 自动加载所有数据表
- ✓ 支持选择表和自定义查询行数
- ✓ 从内存快速查询数据（无需重复连接数据库）
- ✓ 响应式设计，支持大数据量显示
- ✓ 数据统计信息（行数、字段数、查询耗时）
- ✓ SQL注入防护

## 安装和运行

### 1. 安装依赖

```bash
cd db-viewer
pip install -r requirements.txt
```

### 2. 启动服务器

```bash
python db_server.py
```

服务器会在启动时自动连接数据库并缓存数据，然后在 `http://localhost:8888` 启动

### 3. 打开浏览器

访问 `http://localhost:8888` 即可使用

## 使用说明

1. 启动服务器后，所有数据表会自动加载到内存
2. 打开浏览器，页面会自动列出所有数据表
3. 从下拉菜单中选择要查看的表（默认选中第一个）
4. 输入要查询的行数（默认1000，最多10000）
5. 点击"查询数据"按钮，从内存快速获取数据
6. 数据会以表格形式展示，包含所有字段和数据

## 技术栈

- Frontend: HTML + Tailwind CSS + Vanilla JavaScript
- Backend: Python Flask + Flask-CORS
- Database: Tencent CynosDB MySQL
- 缓存: 内存缓存

## 性能优化

- 启动时一次性连接数据库，之后无需重复连接
- 所有表数据缓存到内存，查询速度极快
- 支持页面刷新而无需重新加载数据库

## 注意事项

- 确保Python 3.7+已安装
- 确保数据库连接信息正确
- 启动时会预加载每个表的前1000行数据
- 建议在局域网内使用
